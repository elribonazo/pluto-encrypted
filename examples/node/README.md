# Node Examples

## Prerequisites

- You will need a working mediatorDID, we are currently running a v1 mediator.

## Installation

In this directory, you'll need to run

```bash
npm install
```

## Environment Variables

Set up a `.env` file in this folder, see `.env.example`

```bash
MEDIATOR_DID=
```
